# Welcome to Toko

Toko is your all-in-one toolkit for Revolt servers. Whether you run a community, a project hub, or just a friend group, Toko keeps things smooth, safe, and fun—without the stress.

There’s a **lot** packed in: advanced admin tools, logs, games, tickets, utilities, automations, and more. We’re always adding new features based on what you ask for.

## What Can Toko Do?

Toko 

- **Moderation:** Warnings, bans, mutes, automod, anti-spam, and more
- **Logging:** Track edits, deletes, joins, leaves, and almost everything else
- **Economy:** Server currency, leaderboards, shops, rewards, and games
- **Tickets:** Private channels for support, suggestions, or reports (coming soon)
- **Fun & Games:** Mini-games, polls, interactive commands
- **Utilities:** Timers, reminders, info commands, and a bunch more

and that’s just the start. Explore the [Modules](./modules/utility) section for everything Toko can do.
