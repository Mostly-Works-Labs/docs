# Logging

Toko keeps your server organized by logging all important events automatically!

## Features

- **Message logs:**  
  Tracks deleted and edited messages for moderation and transparency.
- **User actions:**  
  Logs joins, leaves, bans, kicks, and more.
- **Permission changes:**  
  Decodes and logs permission changes with clear, human-readable details.
- **Per-event log channels:**  
  Choose specific channels for each category or event type.
