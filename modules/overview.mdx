## What Each Feature Does

**Economy** - Jobs, gambling, banking, leaderboards  
**Anti-Raid** - Detects and stops server raids automatically  
**Tickets** - React to create private support channels  
**Logging** - Tracks deleted messages, role changes, etc.  
**Fun** - Slap commands, auto-responses, entertainment  

<Warning>
Anti-raid will automatically kick users during raids - configure carefully!
</Warning>
