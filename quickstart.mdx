# Quickstart

Get Toko running on your server in just a few steps!

<Info>
If you get stuck or need help at any point, [join our support server](https://rvlt.gg/48SETK1s)—we’re here for you!
</Info>

<CardGroup cols={2}>
  <Card
    title="1. Invite Toko"
    icon="plus"
    href="https://app.revolt.chat/bot/01JXB622CH7VDC0KFH1VVCC0NJ"
  >
    Add Toko to your Revolt server with a single click.
  </Card>
  <Card
    title="2. Set Permissions"
    icon="shield"
  >
    Make sure Toko can read/send messages, manage channels, and moderate members.<br/>
    <b>Tip:</b> “Administrator” permission is easiest.
  </Card>
  <Card
    title="3. Configure Toko"
    icon="gear"
    href="https://tokobot.xyz/dashboard"
  >
    Optional, but recommended! Set up modules, automod, logging, and more using the dashboard.
  </Card>
  <Card
    title="4. Try a Command"
    icon="terminal"
  >
    In any channel Toko can see, type:<br/>
    <code>/help</code> or <code>/info</code><br/>
    Toko will respond with a list of features and commands.
  </Card>
</CardGroup>
