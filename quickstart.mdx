# Get Toko Running in Minutes

<Warning>
Before you start, please read our <a href="https://terms.tokobot.xyz">Terms of Service</a> and <a href="https://privacy.tokobot.xyz">Privacy Policy</a>.
</Warning>

<Steps>
  <Step title="Invite Toko">
    Add Toko to your Revolt server with a single click.
    
    [Invite Toko](https://app.revolt.chat/bot/01JXB622CH7VDC0KFH1VVCC0NJ)
  </Step>

  <Step title="Set Permissions">
    Toko needs "Manage Server" permission to work with the dashboard.
    
    [See all permissions](#permissions)
  </Step>

  <Step title="Configure Toko">
    Set up modules and features using the dashboard.
    
    [Open Dashboard](https://tokobot.xyz/dashboard)
  </Step>

  <Step title="Try a Command">
    Type `thelp` in any channel to see what Toko can do.
  </Step>
</Steps>

---

## Permissions {#permissions}

**Required:**
- Manage Server (for dashboard access)

**Recommended:**
- View Channel, Send Messages, Send Embeds
- Manage Roles, Kick Members, Ban Members
- Manage Messages, React
